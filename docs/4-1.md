# 4. HTML+CSS+JavaScript
おもにフロントエンド、すなわちブラウザ上で見える部分の知識になります。  

### HTML
Hyper Text Markup Language。  
現在ブラウザで使われるHTMLの仕様はHTML5。

### CSS
Cascading Style Sheet。Webサイトの見た目をつかさどります。

**関連技術やライブラリ**
* SASS
* SCSS
* LESS
* Gulp

### JavaScript
動的な操作や表現が可能になります。ECMAScriptという仕様が存在します。

**関連技術やライブラリ**
* jQuery
* React.js フレームワーク
* Angular.js
* Node.js
* Vue.js
